# RiderShield AI (DashShield) 🛡️

RiderShield AI (DashShield) is a next-generation dashboard designed for seamless protection, intelligent disruption monitoring, and instant claim settlements. The application features a comprehensive premium calculator and dynamic monitoring of various disruption scenarios to keep you informed.

## ✨ Features

- **🛡️ DashShield Coverage:** An intuitive and modern interface to manage your protection plans.
- **🚧 Dynamic Disruption Monitor:** Real-time disruption monitoring with dynamic scenarios that update intelligently.
- **🧮 Smart Premium Calculator:** Flexible premium estimation with both Auto and Manual calculation modes.
- **💸 UPI Integration & Fast Payouts:** Integrated UPI history selection for lighting-fast simulated claim payouts.
- **🎨 Modern UI/UX:** Built with a stunning, responsive design that works flawlessly across all devices.

## 🚀 Step-by-Step Setup Guide

Follow these simple steps to get the project running locally on your machine.

### Prerequisites
Make sure you have installed:
- [Node.js](https://nodejs.org/) (Version 18.0 or higher is recommended)

---

### Step 1: Open the Project Directory
If you have the project downloaded, open your terminal (Command Prompt/Terminal/Mac Terminal) and navigate to the project folder:
```bash
cd path/to/ridershield-ai-main
```

### Step 2: Install Dependencies
Run the following command to download and install all the required packages:
```bash
npm install
```

### Step 3: Start the Server
Once the installation is complete, start the local development server:
```bash
npm run dev
```

### Step 4: View the Application
Open your favorite web browser (Chrome, Safari, Edge) and go to the link shown in your terminal, which is usually:
```text
http://localhost:5173
```

## 🛠️ Technology Stack

This application is built using modern web development technologies:
- **Framework:** React 18 with TypeScript
- **Tooling:** Vite for lightning-fast builds
- **Styling:** Tailwind CSS for beautiful, custom designs
- **Components:** Shadcn UI and Radix UI
- **Icons:** Lucide React

## 📁 Key Components Overview

- `DisruptionMonitor.tsx`: Handles tracking and visualizing different dynamic disruption situations.
- `PremiumCalculator.tsx`: Provides the logic and UI to calculate estimated premiums.
- `UPIPayment.tsx` & `ClaimResult.tsx`: Work together to deliver a seamless payout conclusion.
